Who am I?

I am GDucpm, a:

Sysadmin

I always love to explore new things about Arch Linux, the Linux community and system management. For example, I'm interested in installing Arch Linux, trying out LUKS encryption or install different Linux VMs to learn something new from each Linux distro. I don't fear using hard-to-use distros like Gentoo; but daily driving it is just not ideal.

If someone asks me what kind of sysadmin I am, I'd say I'm a power user. I love taking control over my own PC, whether it's managing services, running scripts, installing new packages, or even delete my entire system (sudo rm -rf --no-preserve-root /, a dangerous command). It might sound scary, but to me, it is the freedom to do ANYTHING when you have all the administrative rights.

Programmer

So far, I've learned a lot of programming languages like Scratch, Python, HTML/CSS/JS, PHP, C, Java, and a bit of C++ or Bash scripting. Through the way, I continue to learn new interesting things; for example, when I finished the basics of Java and start to learn C++, I notice interesting similarities like the OOP philosophy and differences such as the lack of true pointers in Java (for safety reasons).

Every programming language I learn builds skill and confidence. Over the years, with basic C knowledge, I managed to create some very simple programs, but they got rejected on the Arch User Repository (AUR). That's fine. I'm not someone with advanced C knowledge. But if it was someone who wanted simple HTML webpages, I'd shine.

Philosopher

I support the movement of Free Software (free in terms of freedom, not in terms of price) and the Four freedoms: to be able to run code, to be able to modify the source code, to be able to distribute it and to be able to distribute your own modified versions. Basically, this gives fairness to both users and whatever company that sold the product to you. It doesn't have to be $0; it just has to be freely distributable and the source code has to be open to the public.

For those who do not know, free/open-source software gives more advantages than most people think. Any team of developers can inspect a free application for malicious functionalities, or lure out anti-user features and redistribute it for you. People can also share and improve code. But the originator goes to the Free Software Foundation led by Richard Matthew Stallman. For even more info, see the book: "Free software, Free society" by Richard Stallman.

I became a sysadmin because I heard a lot about Linux and then I decided to research about it. When I asked ChatGPT I was able to learn about it. What do you think about unethical social media platforms

Whoa, that's a weird idea.

Facebook is a world-popular social media aimed at all ages, providing an engaging social media experience. However, I must say I oppose it for ethical reasons. Facebook has massive data collection, for example, they've built an ecosystem where millions of websites on the Internet send interaction usage to Facebook (a.k.a. Meta Pixel), and other awkward tracking patterns like deleted posts or liked posts. After all, I see this as unethical. It normally is okay with only a bit of tracking, but this is a completely different story. It made me extremely concerned. They harvest our data and make billions. But I don't care! They're out there hosting a social media just for their greed. As a philosopher, I don't want to do unethical stuff even if I make much.

If you share the same thought, you can always lend a hand. What if you simply tell people to stop using Facebook? I appreciate any help to stop this evil social media. Thanks for hearing my thoughts about this.

Fun fact: The term for this is "surveillance capitalism".
